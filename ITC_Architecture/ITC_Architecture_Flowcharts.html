4. Full Logic Pipeline

flowchart TD
    subgraph S2[Step 2 - Expenses]
        A1[File 4: Working All Type sheet] --> A2[Categorize rows]
        A2 --> A3[CON = GSTIN + Bill No]
        A3 --> A4[Ineligible Debit only]
    end
    subgraph S1[Step 1 - MRR]
        B1[Files 1+2 CSV] --> B2[Combine + Negate returns]
        B2 --> B3[Map Voucher from Step 2]
        B3 --> B4[Pivot deduplicate]
    end
    subgraph S3[Step 3 - Books]
        C1[Merge MRR + Expenses] --> C2[Skip duplicate CON]
        C2 --> C3[Aggregate by CON]
    end
    subgraph S4[Step 4 - GSTR-2B]
        D1[Match CON key] --> D2[Compare taxes]
        D2 --> D3[Assign Remarks]
    end
    subgraph S5[Step 5 - Stock]
        E1[Stock received] --> E2[GIT mapping]
        E2 --> E3[TAKEN IN 3B]
    end
    S2 --> S1 --> S3 --> S4 --> S5

5. CON Key Match Logic

flowchart TD
    ROW[Books Row] --> K[CON = GSTIN + Invoice]
    K --> S{Found in GSTR-2B?}
    S -->|No| R1[Not in 2B]
    S -->|Yes| T{Tax match?}
    T -->|All match| R2[PERFECT MATCHED]
    T -->|Partial| R3[PROBABLE MATCHED]
    T -->|Different| R4[Mismatched]
